# Legal GC Brain Examples

This directory demonstrates how GBrain can be used as a company-wide legal and governance memory layer.

## Workflow Catalog

### Product Launch Review

Inputs:
- feature description
- architecture notes
- DPA
- privacy policy
- AI governance policy

Outputs:
- risk summary
- missing information
- escalation requirements
- launch recommendation

### Vendor and DPIA Review

Inputs:
- vendor questionnaire
- security review
- DPA
- subprocessors

Outputs:
- risk rating
- DPIA recommendation
- remediation actions
- approval status

### AI Agent Governance

Inputs:
- proposed agent capability
- tool permissions
- data sources
- human oversight model

Outputs:
- governance classification
- required controls
- monitoring requirements
- approval workflow

## Example query

What legal, privacy and AI governance concerns should be reviewed before enabling an autonomous customer-facing AI agent?
